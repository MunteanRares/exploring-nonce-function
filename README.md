# Exploring Nonce Function in Blockchain

This is a simple and basic representation of an implementation in C# with mining, blockchain validation and proof-of-work (nonce).

## Table of Contents
 - [Console App]: It includes a console app where you can visualize the blocks, the nonce adn the time it took to mine it.
 - [.Core]: Here you can find the implemenetation of the Block class and Blockchain class

## Installation
1. Clone the repository: `https://github.com/MunteanRares/exploring-nonce-function.git`
2. Open the project in Visual Studio.
3. Build the solution.
